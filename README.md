## About

Pick colors from anywhere on screen with precision.

## Features

- Real-time color picking
- Multiple color formats
- Magnifier
- Color history

## Installation

### From App Store
Open the **App Store** in zOS and search for "Digital Color Meter".

### Via npm
```bash
npm install @zos-apps/digital-color-meter
```

## Usage

- Move cursor to sample colors
- Click to copy color value
- Choose format from dropdown



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
